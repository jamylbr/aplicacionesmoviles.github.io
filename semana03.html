Proyecto 01: Área y Volumen de un Cilindro

Descripción: Diseñe un programa en Dart para calcular el área total y volumen de un cilindro dado su radio y altura.


import 'dart:math';

void main() {
  double r = 5.0;
  double h = 10.0;
  double area = 2 * pi * r * (r + h);
  double volumen = pi * pow(r, 2) * h;
  print('El área total del cilindro es: $area');
  print('El volumen del cilindro es: $volumen');
}

Proyecto 02: Repartición de Dinero entre Hijos

Descripción: Diseñe un programa en Dart que determine la cantidad de dinero que recibirá cada hijo según las proporciones dadas.


void main() {
  double totalDinero = 1000.0;
  double josue = 0.27 * totalDinero;
  double daniel = 0.25 * totalDinero;
  double tamar = 0.85 * josue;
  double caleb = 0.23 * (josue + daniel);
  double david = totalDinero - (josue + daniel + tamar + caleb);
  print('Josué recibe: \$${josue.toStringAsFixed(2)}');
  print('Daniel recibe: \$${daniel.toStringAsFixed(2)}');
  print('Tamar recibe: \$${tamar.toStringAsFixed(2)}');
  print('Caleb recibe: \$${caleb.toStringAsFixed(2)}');
  print('David recibe: \$${david.toStringAsFixed(2)}');
}

Proyecto 03: Distribución de Inversión en Rubros

Descripción: Diseñe un programa en Dart que determine cuánto dinero se invertirá en cada rubro de una feria según el porcentaje asignado.


import 'dart:io';

void main() {
  print('Ingrese el monto total de inversión:');
  double? totalDinero = double.parse(stdin.readLineSync()!);
  double alquiler = 0.23 * totalDinero;
  double publicidad = 0.07 * totalDinero;
  double transporte = 0.26 * totalDinero;
  double servicios = 0.12 * totalDinero;
  double decoracion = 0.21 * totalDinero;
  double gastosVarios = 0.11 * totalDinero;
  print('Gasto en alquiler de espacio: \$${alquiler.toStringAsFixed(2)}');
  print('Gasto en publicidad: \$${publicidad.toStringAsFixed(2)}');
  print('Gasto en transporte: \$${transporte.toStringAsFixed(2)}');
  print('Gasto en servicios feriales: \$${servicios.toStringAsFixed(2)}');
  print('Gasto en decoración: \$${decoracion.toStringAsFixed(2)}');
  print('Gasto en gastos varios: \$${gastosVarios.toStringAsFixed(2)}');
}
